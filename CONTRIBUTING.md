# Contributing to GenAI Telemetry

## Reporting Bugs
If you encounter a bug, please file an issue with clear reproduction steps and any potential solutions. Pull requests with fixes are also welcome.
## Working on Existing Issues
Browse open issues and contribute by answering questions, reproducing problems, suggesting solutions, or submitting pull requests.
Requesting Features
For new features or major enhancements, open an issue explaining the proposed change and its rationale. Pull requests implementing new features are welcome.
## Reviewers & Approvers
genai-telemetry reviewers and approvers participate in the PR review process and help guide newcomers through their first contributions.
Becoming a Contributor

Submit a pull request to get started. All contributions must be accompanied by acknowledgment of and agreement to the [Developer Certificate of Origin (DCO)](https://developercertificate.org/). Submissions are reviewed as quickly as possible.
