## Responsibilities

### Lead Maintainer
- Set project direction and priorities
- Final decision authority on technical matters
- Represent project externally
- Manage releases

### Maintainers
- Review and merge pull requests
- Triage issues
- Mentor contributors
- Participate in governance decisions

### Committers
- Review pull requests
- Commit code changes
- Help maintain code quality
- Support contributors
